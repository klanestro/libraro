The website

If you are a Python programmer familiar with the Django framework, you can contribute to the project on github. All changes are committed to the github project and then pulled to my hosting account. The wiki on github contains information on how to set up a development environment.

Please email me to report problems and share suggestions.

The content

This website needs a “Learn” section, containing a concise collection of articles on Esperanto’s grammar. It is not a top priority for me, so I would be glad if someone gave me a hand creating it. If you send me articles, I will format them and put them online. Found, written from scratch or modified—they have to be in the public domain or licenced to be free to use both non-commercially and commercially.

If you would like a certain text to appear on this website, please email me with a link or an included file. The preferred formats are HTML and FB2.

Donate

This website is free, its source code is open, it has no advertisements. It is a product of many hours of work. If you found this website useful, you can make a donation using PayPal here.
